# SGH - Documentación del Proyecto

Este repositorio contiene la documentación completa del proyecto **SGH (Sistema de Gestión de Horarios)**, un sistema integral para la administración de horarios en instituciones educativas.

## Estructura de la Documentación

- **00-Presentacion_Del_Equipo_SGH.pptx**: Presentación del equipo de desarrollo.
- **01-Especificacion de requisitos/**: Especificación de Requisitos Software (SRS) del sistema.
- **02-Analisis de software/**: Diagramas de análisis (actividades, casos de uso, clases, flujos, secuencias).
- **03-Propuesta técnica/**: Propuesta técnica con tecnologías y arquitectura.
- **04-Diseño de software/**: Diseño de la interfaz (enlaces a Figma).
- **05-Codigo/**: Enlace al repositorio del código fuente.
- **06-Pruebas/**: Reportes de pruebas unitarias (backend, frontend web, frontend móvil).
- **07-Implementacion de software/**: Manual de implementación del sistema.
- **08-Correcciones/**: Informe de calidad del software.
- **09-Manuales/**: Manuales de usuario (administrador, móvil).
- **Soportes/**: Documentos de soporte (entrevistas, fichas técnicas, matrices de riesgos, etc.).

## Tecnologías Utilizadas

- **Backend**: Java con Spring Boot, MySQL.
- **Frontend Web**: Next.js (basado en React), Tailwind CSS.
- **Frontend Móvil**: React Native con Expo.
- **Base de Datos**: MySQL.
- **DevOps**: Docker, Jenkins.
